# Redesign Projekt (Figma)

Beschreibung:
Dieses Projekt umfasst das Redesign der Website Cranach Digital Archive. Ziel war es, die Benutzeroberfläche moderner und übersichtlicher zu gestalten, um die Benutzerfreundlichkeit zu verbessern. Es wurden die Startseitte sowie drei Inhaltsseiten neu gestaltet.

Seiten:
- Startseite
- Aktuelles
- Lucas Cranach Inhaltsseite
- Forschung

Tools:
- Figma
